This repository contains data and analysis scripts accompanying the manuscript. It is organised into two subfolders: exp1 (Experiment 1, fMRI) and exp2 (Experiment 2, TUS).
MATLAB (.m) files are used for data preprocessing. Behavioural analyses are performed in R and provided as R Markdown (.Rmd) files, which should be run in RStudio. Please install the necessary R packages before running the R Markdown files. All datasets can be found in the same folder as the corresponding analysis scripts. Expected outputs correspond to the results reported in the main text of the manuscript.

Data availability:
Behavioural data and analysis scripts are available at https://github.com/shuyil/causalInf. 
Whole-brain statistical images are available at https://neurovault.org/collections/HIIMHDBP/. 
Neuroimaging data are available from the corresponding author upon reasonable request.
